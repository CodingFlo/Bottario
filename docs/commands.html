Twitch Bot-Befehle

Bot-Prefix: #
Befehl Kürzel Beschreibung
agb - Ein Spaßbefehl mit dem man darüber aufklärt, dass Chris gerade Unwahrheiten erzählt, aber trotzdem Recht hat laut seinen AGBs
chatlock - Ein Informationsbefehl mit dem man darüber aufklärt wird um was es sich bei einer Chatlock handelt
chatshop - Ein Befehl der mehr Informationen bezüglich der Chatlock (!chatlock) gibt
cheater - Ein Spaßbefehl mit dem man Chris darüber aufmerksam macht, dass er cheatet
discord dc Ein Befehl mit dem man den Link zu Chris seinem Discord-Server bekommt
followage - Ein Befehl mit dem angezeigt wird, wie lange ein User Chris schon folgt, bzw. ob ein User Chris folgt
freundescode fc Ein Befehl mit dem man den Switch Freundescode von Chris bekommt
insta - Ein Befehl mit dem man den Link zu Chris seinem Instagram-Account bekommt
lurk - Ein Befehl mit dem man Chris mitteilen kann, dass man in den Lurk-Modus wechselt
of - Ein Spaßbefehl mit dem man den Link zu Chris seinem Discord-Server bekommt
peter - Ein Spaßbefehl mit dem aufgeklärt wird, warum Chris oft Peter schreit oder es nicht böse gemeint ist
setgame game Ein Befehl rein für Mods, mit dem das gestreamte Game geändert werden kann
settitle title Ein Befehl rein für Mods, mit dem der Streamtitel geändert werden kann
shoutout so Ein Befehl rein für Mods um jemanden einen Shoutout zu geben
spoilertario spoiler Ein Spaßbefehl mit dem man darüber aufklärt, dass Chris wieder etwas gespoilert hat.
streik - Ein Spaßbefehl, der ein Zeichen setzt
test - Ein Befehl mit dem man den testen kann, ob der Bot auf Befehle reagiert oder ein Fehler vorliegt
tiktok tt Befehl mit dem man den Link direkt zu Chris' TikTok Kanal bekommt
twitter x Befehl mit dem man den Link direkt zu Chris' X/Twitter Kanal bekommt
unlurk - Befehl mit dem man mitteilen kann, dass man nun nicht mehr lurkt
vod - Befehl mit dem man den Link direkt zu Chris' VOD Kanal bekommt
yta - Befehl mit dem man den Link direkt zu Chris seinen Anno Kanal bekommt
yt - Befehl mit dem man den Link direkt zu Chris' YouTube Kanal bekommt
yth - Befehl mit dem man den Link direkt zu Chris seinen Horror YouTube Kanal bekommt
yto - Befehl mit dem man den Link direkt zu Chris' seinen Pokemon Opening Kanal bekommt
info - Ein Befehl, der einige Informationen bezüglich Bottario gibt.
commands command Ein Befehl, der alle verfügbaren Befehle auflistet.
timermessages tm Ein Befehl, der alle verfügbaren Timer-Nachrichten auflistet.
